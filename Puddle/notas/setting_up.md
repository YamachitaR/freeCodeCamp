# Setting Up

Instalando ambiente virtual 

~~~ bash
python3 -m venv env 
~~~

Ativando Ambiente Virtual 


~~~ bash
source env/bin/activate 
~~~


Instalando Django 


~~~ bash
pip install django
~~~

# Criando o Projeto 

~~~bash
django-admin startproject puddle . 
~~~

 
 # Roando o projeto 

~~~ bash 
python3 manage.py runserver
~~~
